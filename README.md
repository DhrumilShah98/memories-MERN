# memories-MERN
Memories - A MERN application

A Full Stack MERN application (MongoDB, Express, React and Node).
The application is called "Memories" and it is a simple social media app that allows users to post interesting events that happened in their lives.

Youtube Series: [JavaScript Mastery Videos](https://www.youtube.com/watch?v=ngc9gnGgUdA&list=PL6QREj8te1P7VSwhrMf3D3Xt4V6_SRkhu)

Setup:<br/>
run `npm i && npm start` for both client and server side to start the app
